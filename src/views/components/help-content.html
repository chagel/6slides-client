Get the Most Out of Notion to Slides

Learn how to use the extension to create beautiful presentations from your Notion pages.

Instructions

Your Notion page must use this structure:

  • H1 (#) elements start new slides
  • H2 (##) elements are section titles
  • H3 (###) elements are subsections
  • Content between H1s belongs to the previous H1
  • Lines starting with - or * become bullet points
  • Regular text becomes paragraphs

Example Format

# Slide 1 Title This paragraph will appear on slide 1. - This is a bullet point on slide 1 - Another bullet on slide 1 ## Section on Slide 1 More content for slide 1... # Slide 2 Title This paragraph will appear on slide 2. - Bullet on slide 2 ``` // Some code block function helloWorld() { console.log("Hello from slide 2!"); } ``` # Slide 3 Title > This is a quote block on slide 3 ### Subsection on Slide 3 Final slide content...

Supported Notion Blocks

  • Code blocks
  • Quote blocks
  • Images (with captions)
  • Tables
  • Dividers (horizontal rules)
  • To-do and toggle lists
  • Others

How to Use

  1. Navigate to a Notion page
  2. Click the extension icon
  3. Click "Present"
  4. The slides will open in the current tab

Frequently Asked Questions

Why isn't my Notion page converting correctly?

Make sure your Notion page follows the template format:

  • Each H1 heading creates a new slide
  • You need at least one H1 heading to generate slides
  • Check if you have proper permissions for the Notion page
How do I customize my presentation?

You can customize your presentation in several ways:

  • Visit the Settings page to change the theme, transitions, and more
  • Use Markdown formatting in your Notion page for text styling
  • Add images and other rich content in Notion
Is my data secure?

Yes! Notion to Slides processes everything locally in your browser. No data is sent to external servers or stored anywhere except in your browser's local storage.
