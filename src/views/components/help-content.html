Instructions

Your Notion page must use this structure:

  • H1 (#) elements start new slides
  • H2 (##) elements create vertical subslides
  • H3 (###) elements are subsections within a slide
  • Content between H1s belongs to the previous H1
  • Lines starting with - or * become bullet points
  • Regular text becomes paragraphs

Check out this Notion page for a detailed example.

Supported Notion Blocks

  • Code blocks
  • Quote blocks
  • Images
  • Tables
  • Bullet points and numbered lists
  • Others

How to Report Issues

If you encounter any problems or have suggestions for improving Six Slides:

Option 1: Email Support

  1. Email us at support@6slides.com with details about the issue(and your subscription account if any)
  2. Include steps to reproduce the problem if possible
  3. Mention which browser and version you're using
  4. For feature requests, describe the functionality you'd like to see

Option 2: Use Debug Mode

  1. Go to the About page and enable Debug Mode in the Developer section
  2. Reproduce the issue you're experiencing
  3. Click the expand button on the sidebar to view debug logs
  4. Click the "Report" button to automatically send logs and issue details

We value your feedback and are committed to making Six Slides better with each update!
